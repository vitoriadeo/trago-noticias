Política de privacidade

Última atualização: 18 de outubro de 2025

Este projeto respeita a privacidade e a proteção dos dados pessoais dos usuários, conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD).

Dados coletados

Ao utilizar este site, podemos coletar: nome (para personalizar a comunicação) e endereço de e-mail (para envio de alertas de notícias).

Finalidade da coleta

Os dados coletados serão usados exclusivamente para enviar alertas sobre notícias relacionadas aos termos cadastrados pelo usuário.

Base legal

O tratamento dos dados é realizado com base no consentimento do titular, concedido no momento do cadastro.

Compartilhamento de dados

Não compartilhamos seus dados com terceiros, exceto quando exigido por lei.

Armazenamento e segurança

Os dados são armazenados em banco de dados seguro, com acesso restrito e medidas de proteção contra acesso não autorizado.

Prazo de retenção dos dados

Os dados serão mantidos enquanto o usuário mantiver seu cadastro ativo ou até a solicitação de exclusão, o que ocorrer primeiro.

Direitos do titular

Você pode, a qualquer momento: solicitar acesso aos seus dados, solicitar a correção ou exclusão e revogar o consentimento.

Canal de contato

Responsável pelo tratamento de dados: Vitória de Oliveira. O e-mail para solicitações é [contato@tragonoticias.site] ou acesse a aba 'Contato' no menu abaixo.

Alterações nesta política

Podemos atualizar este documento para melhor atender à legislação ou melhorias no serviço. Recomendamos verificar esta página periodicamente.
